# Sleep Tracker

A simple, offline-first sleep tracking Progressive Web App built with React and TypeScript.

## Goals

- Simple to use
- Low battery usage
- Works offline
- Cross-platform
- Account syncing in future versions

## Current Features (v0.1)

- Sleep tracking
- Sleep history
- Weekly averages
- Settings
- Dark mode
- Local storage
